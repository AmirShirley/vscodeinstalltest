# Kubuntu VS Code Installer

A simple bash script to automate the installation of Microsoft Visual Studio Code on Kubuntu.

## What This Script Does

- Updates your package manager
- Installs required dependencies
- Adds Microsoft's official VS Code repository
- Installs the latest version of VS Code
- Provides clear feedback at each step
